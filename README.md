# titan
Blazingly fast scraper

# Install

#### Linux/Mac
- First of all install rust then
````
git clone 'https://github.com/Based-Programmer/titan' && \
cd titan && \
cargo build --release
````

- Then add it to yout $PATH
````
sudo cp target/release/titan /usr/local/bin/
````

- Usage
````
titan <argument> <url>
````
